Simple Weather widget, that displays current prevaling weather conditions.

== Description ==

Simple Weather widget, that displays current weather based on provided woeid.
The location parameter needs to be a WOEID. To find WOEID, browse or search from http://weather.yahoo.com. 
WOEID is in the link for the forecast page.You can also get the WOEID by zip-code.
 

== Installation ==

1. Install the Plugin.
2. Login to Wordpress Admin and activate the plugin.
